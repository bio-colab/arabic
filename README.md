# محلل تكرار الكلمات العربية (Arabic Text Analyzer)

تطبيق ويب متقدم مبني باستخدام React و TypeScript لتحليل النصوص العربية واستخراج الإحصائيات اللغوية والأسلوبية. يعمل التطبيق بالكامل داخل المتصفح (Client-Side) لضمان أقصى درجات الخصوصية والسرعة، دون الحاجة لرفع النصوص إلى أي خوادم خارجية.

## الميزات الرئيسية

1. **تحليل نص واحد**:
   - استخراج الكلمات الأكثر تكراراً مع حساب النسبة المئوية.
   - دعم تحليل الكلمات المفردة (Unigram)، والثنائيات (Bigram)، والثلاثيات (Trigram).
   - إمكانية إزالة كلمات التوقف (Stopwords) والسوابق (ال، و، ب، ف، ك، ل).
   - تجميع الكلمات حسب الجذر (Stemming) باستخدام محرك جذور خفيف مدمج.
   - عرض النتائج في جدول تفاعلي وسحابة كلمات (Word Cloud).

2. **مقارنة نصوص**:
   - مقارنة نصين مختلفين لاكتشاف الفروق الدقيقة في استخدام المفردات.
   - ترتيب النتائج حسب أكبر فرق في التكرار (Delta).

3. **تحليل عبر الزمن**:
   - تتبع تطور استخدام كلمات معينة عبر سلسلة من النصوص (مثل فصول رواية أو مقالات متسلسلة).
   - رسم بياني خطي يوضح صعود وهبوط استخدام المفردات.

4. **لوحة تحكم متقدمة (Advanced Dashboard)**:
   - مقارنة نصوص متعددة (Corpus) من عدة أبعاد.
   - **البُعد الكمّي**: إجمالي الكلمات، التنوع اللفظي (Lexical Diversity)، ومتوسط طول الجملة.
   - **البُعد الأسلوبي**: تقدير أقسام الكلام (أسماء، أفعال، حروف) باستخدام خوارزميات استدلالية.
   - **البُعد الموضوعي**: استخراج الكلمات المفتاحية المميزة لكل نص باستخدام خوارزمية TF-IDF.

## التقنيات المستخدمة

- **React 19**: لبناء واجهة مستخدم تفاعلية وسريعة.
- **TypeScript**: لضمان جودة الكود وتجنب أخطاء وقت التشغيل.
- **Tailwind CSS**: لتنسيق الواجهة ودعم الوضع الليلي (Dark Mode).
- **Recharts**: لرسم المخططات البيانية التفاعلية.
- **Web Workers**: لمعالجة النصوص الكبيرة في الخلفية دون تجميد واجهة المستخدم.
- **Mammoth.js**: لقراءة واستخراج النصوص من ملفات Word (.docx).

## التثبيت والتشغيل

1. قم بتثبيت الحزم المطلوبة:
   ```bash
   npm install
   ```

2. قم بتشغيل خادم التطوير:
   ```bash
   npm run dev
   ```

3. افتح المتصفح على العنوان `http://localhost:3000`.

## الخصوصية والأمان

هذا التطبيق يعمل بالكامل في المتصفح (Client-Side). لا يتم إرسال أي نصوص أو بيانات إلى أي خادم خارجي. جميع عمليات المعالجة والتحليل تتم على جهاز المستخدم.
